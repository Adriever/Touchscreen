# Touchscreen
A program that reads and displays a touchscreens value based on where you touch, as well as turns an LED on the further up the x-axis you get
Added LED that would take the x value reading and turn on an LED based on that value, changing the leds brightness as you slide across the touchscreen
changed x to -100 and /4 to get the LED to turn off when at the end of the touchscreen and to be full at the other end of the screen
